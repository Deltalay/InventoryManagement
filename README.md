# Inventory Management System

## Setup

Create a virtual enviroment

```shell
python -m venv .venv
```

Activate the virtual enviroment
For Windows(Powershell):

```shell
./.venv/Scripts/Activate.ps1
```

For Windows(CMD):

```shell
cd .venv/Scripts
activate
```

For Linux:

```bash
source .venv/bin/activate
```
